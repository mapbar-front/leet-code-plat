# template-vue-pc

> 一个低代码搭建平台

## 使用

```bash
npm install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and hot-reloads for mock development

```
npm run mock
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
